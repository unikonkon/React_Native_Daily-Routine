สิ่งที่จะเขียนใน React Native

  • YearScreen — FlatList เดือน (MiniMonth) ข้ามปี พ.ศ. ไฮไลต์เดือนปัจจุบัน แตะ → เดือน
  • MonthScreen — grid 7×6 + จุดสีกิจกรรม (group ตามวันที่) แตะวัน → ไทม์ไลน์
  • DayScreen — แถบสัปดาห์ + ไทม์ไลน์ 24 ชม. + เส้นเวลาปัจจุบัน (auto-scroll)
  • Shared — selectedDate/viewMode state, ปุ่ม “วันนี้”, +เพิ่ม, ค้นหา, locale ไทย +543 ปี
